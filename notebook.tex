    \section{Quora Duplicate Questions Checker

    This is the submission for a task on flagging duplicate questions on the
Quora dataset from Kaggle.

    \subsection{The Approach}\label{the-approach}

The questions will be converted to respective feature vectors using
\textbf{\emph{doc2vec}} (gensim's implementation). Then a
\textbf{\emph{Siamese network}} will be trained to predict the duplicacy
of questions. The code with all the explanation is given below.
